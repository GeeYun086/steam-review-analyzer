# Product Brief

## 1. Product Summary

이 서비스는 게임회사 사업팀·마케팅팀 및 인디 게임 개발자가 Steam 유저 리뷰를 수동으로 읽고 정리하는 반복 작업을 자동화하도록 돕는다. 게임 이름을 검색하면 AI가 리뷰 수집 → 카테고리 분류 → 개선 피드백 생성까지 전 과정을 자동으로 처리하여, "어떤 부분을 개선하면 평점이 올라갈지" 액션 가능한 인사이트를 즉시 제공한다.

## 2. Problem

게임 출시 후 유저 리뷰는 수백~수천 개가 쌓이지만, 담당자가 직접 읽고 분류하는 데만 수 시간이 걸린다. 부정 리뷰가 많아도 어떤 카테고리(그래픽, 최적화, 스토리 등)에서 문제가 집중되는지 구조적으로 파악하기 어렵다. 엑셀이나 수작업 정리는 비효율적이고, 분석 결과를 팀에 공유할 형태로 만들기까지 추가 작업이 필요하다. 결국 중요한 유저 피드백이 실제 개선 우선순위에 반영되지 못하는 경우가 많다.

## 3. Target User

- **주요 사용자:** 게임회사 사업팀·마케팅팀, 인디 게임 개발자
- **사용 상황:** 자사 게임 업데이트 후 유저 반응을 점검할 때, 경쟁 게임을 분석할 때, 개선 우선순위를 팀 내에서 논의할 때
- **사용자의 기술 수준:** 별도 설치 없이 웹 브라우저에서 바로 사용 가능한 수준을 기대함
- **사용자가 기대하는 결과:** 게임 이름 검색 한 번으로 카테고리별 감성 분포와 구체적인 개선 제안을 즉시 확인

## 4. Core Value

수백 개의 Steam 리뷰를 AI가 자동으로 수집·분류하고 개선 우선순위까지 제시하여, 담당자가 수 시간의 수작업 없이 바로 의사결정에 집중할 수 있게 한다.

## 5. MVP Scope

이번 MVP에 포함할 것:

- **게임 검색:** 게임 이름 입력 시 Steam 게임 목록 표시, 선택 시 기본 정보(장르, 출시일, 현재 평점) 표시
- **리뷰 자동 수집:** Steam Web API로 최신 리뷰 최대 100개 자동 수집 (긍정 50 / 부정 50)
- **AI 감성 분석:** 리뷰를 7개 카테고리(게임플레이, 그래픽, 사운드, 스토리, 최적화, 가격, 멀티플레이)로 자동 분류 및 긍정·부정 비율 계산
- **개선 피드백 생성:** 핵심 문제 요약 + 개선 우선순위 리스트(1~3위) + 카테고리별 대표 유저 발언 인용
- **대시보드 시각화:** 카테고리별 감성 도넛 차트 + 개선 우선순위 카드 목록

## 6. Non-goals

이번 MVP에서 제외할 것:

- 로그인 / 회원가입
- 결제
- 여러 게임 동시 비교 분석
- 시계열 트렌드 분석 (업데이트 전후 비교)
- 한국어 / 영어 언어 필터
- 분석 결과 PDF 내보내기
- 분석 히스토리 저장

## 7. Success Criteria

- 사용자는 게임 이름 검색부터 분석 결과 확인까지 1분 안에 완료할 수 있다.
- 3회차 종료 시 브라우저에서 검색 → 수집 → 분석 → 대시보드 전체 흐름을 확인할 수 있다.
- 4회차에서 핵심 흐름을 Playwright 테스트로 검증할 수 있다.
- 최종 결과물은 Vercel에 배포 가능한 형태여야 한다.
