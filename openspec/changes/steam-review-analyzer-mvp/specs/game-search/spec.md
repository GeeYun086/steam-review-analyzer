## ADDED Requirements

### Requirement: 게임 검색
시스템은 사용자가 입력한 게임 이름으로 Steam 스토어를 검색하고, 매칭된 게임 목록(이름, appId, 헤더 이미지 URL)을 반환해야 한다(SHALL).

#### Scenario: 유효한 게임명 검색
- **WHEN** 사용자가 유효한 게임명(예: "Elden Ring")을 입력하고 검색을 실행하면
- **THEN** 시스템은 최대 5개의 매칭 게임 목록을 반환한다

#### Scenario: 검색 결과 없음
- **WHEN** 사용자가 존재하지 않는 게임명을 입력하면
- **THEN** 시스템은 빈 배열과 함께 적절한 메시지를 반환한다

#### Scenario: 빈 쿼리 검색
- **WHEN** 사용자가 빈 문자열로 검색을 시도하면
- **THEN** 시스템은 400 에러를 반환하고 검색을 실행하지 않는다

### Requirement: 게임 기본 정보 표시
시스템은 선택된 게임의 기본 정보(장르, 출시일, 전체 평점, 리뷰 수)를 Steam Store API에서 가져와 표시해야 한다(SHALL).

#### Scenario: 게임 선택 후 정보 표시
- **WHEN** 사용자가 검색 결과에서 게임을 선택하면
- **THEN** 시스템은 해당 게임의 appId로 기본 정보를 조회하여 표시한다

#### Scenario: 정보 조회 실패
- **WHEN** Steam API가 게임 정보를 반환하지 않거나 오류가 발생하면
- **THEN** 시스템은 오류 메시지를 표시하고 재시도 옵션을 제공한다
