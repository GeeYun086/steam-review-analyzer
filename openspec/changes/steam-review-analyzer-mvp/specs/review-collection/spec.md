## ADDED Requirements

### Requirement: 리뷰 자동 수집
시스템은 지정된 appId에 대해 Steam Web API에서 최대 100개(긍정 50개, 부정 50개)의 리뷰를 자동으로 수집해야 한다(SHALL).

#### Scenario: 정상 리뷰 수집
- **WHEN** 유효한 appId로 리뷰 수집 요청이 들어오면
- **THEN** 시스템은 긍정 리뷰 최대 50개, 부정 리뷰 최대 50개를 합산하여 반환한다

#### Scenario: 리뷰 100개 미만 게임
- **WHEN** 게임의 전체 리뷰 수가 100개 미만인 경우
- **THEN** 시스템은 실제 수집 가능한 수량만큼만 반환하고 분석을 계속 진행한다

#### Scenario: 리뷰가 없는 게임
- **WHEN** 게임에 리뷰가 전혀 없는 경우
- **THEN** 시스템은 빈 배열을 반환하고 분석 불가 메시지를 표시한다

### Requirement: 리뷰 데이터 정제
시스템은 수집된 리뷰에서 분석에 필요한 필드(리뷰 본문, 추천 여부, 작성 시간)만 추출하여 반환해야 한다(SHALL). 리뷰 본문이 비어있거나 10자 미만인 리뷰는 제외한다.

#### Scenario: 짧은 리뷰 필터링
- **WHEN** 리뷰 본문이 10자 미만인 경우
- **THEN** 해당 리뷰는 분석 대상에서 제외된다

#### Scenario: 정상 리뷰 반환
- **WHEN** 유효한 리뷰를 수집하면
- **THEN** 각 리뷰는 { text, recommended, timestamp } 형식으로 반환된다
