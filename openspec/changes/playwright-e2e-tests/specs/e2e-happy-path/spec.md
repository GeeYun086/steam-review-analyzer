## ADDED Requirements

### Requirement: 게임 검색 후 목록 표시
시스템은 사용자가 게임명을 입력하고 검색하면 Steam API를 통해 검색 결과 목록을 화면에 표시해야 한다(SHALL).

#### Scenario: 유효한 게임명으로 검색
- **WHEN** 사용자가 검색창에 "Elden Ring"을 입력하고 검색 버튼을 누르면
- **THEN** 시스템은 1개 이상의 게임 카드를 화면에 표시한다

#### Scenario: 검색 결과에서 게임 선택
- **WHEN** 사용자가 검색 결과 목록에서 특정 게임 카드를 클릭하면
- **THEN** 해당 게임이 선택 상태(selected)로 표시되고 "리뷰 분석 시작" 버튼이 나타난다

### Requirement: 분석 실행 및 대시보드 라우팅
시스템은 사용자가 분석 시작 버튼을 클릭하면 리뷰 수집과 AI 분석을 순서대로 실행하고 대시보드 페이지로 이동해야 한다(SHALL).

#### Scenario: 분석 완료 후 대시보드 이동
- **WHEN** 사용자가 게임을 선택한 뒤 "리뷰 분석 시작" 버튼을 클릭하면
- **THEN** 로딩 스피너가 표시되고, 분석 완료 후 `/dashboard?appId=<appId>` 페이지로 이동한다

#### Scenario: 대시보드에서 분석 결과 표시
- **WHEN** 대시보드 페이지(`/dashboard?appId=<appId>`)가 로드되면
- **THEN** AI 요약 텍스트, 카테고리별 차트 7개, 개선 우선순위 카드가 화면에 표시된다

#### Scenario: 다른 게임 검색 버튼으로 홈 복귀
- **WHEN** 대시보드에서 "다른 게임 검색" 버튼을 클릭하면
- **THEN** 시스템은 홈(`/`) 페이지로 이동한다
