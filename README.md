# Steam Review Analyzer

> Steam 게임 이름을 검색하면 AI가 유저 리뷰를 자동 수집·분류하고, 어떤 부분을 개선하면 평점이 올라갈지 액션 가능한 피드백을 생성해주는 웹 대시보드.

---

## 소개

게임 출시 후 쌓이는 수백 개의 Steam 리뷰를 담당자가 직접 읽고 정리하는 작업을 자동화합니다.  
게임 이름 검색 한 번으로 리뷰 수집 → 카테고리 분류 → 개선 우선순위 생성까지 전 과정을 처리합니다.

**대상 사용자**
- 게임회사 사업팀 / 마케팅팀 — 자사 게임 유저 반응 모니터링
- 인디 게임 개발자 — 경쟁 게임 분석, 자기 게임 피드백

---

## 주요 기능

- **게임 검색** — Steam에서 게임 이름으로 검색, 기본 정보(장르·출시일·평점) 표시
- **리뷰 자동 수집** — Steam Web API로 최신 리뷰 최대 100개 수집 (긍정 50 / 부정 50)
- **AI 감성 분석** — 7개 카테고리(게임플레이, 그래픽, 사운드, 스토리, 최적화, 가격, 멀티플레이)로 자동 분류
- **개선 피드백 생성** — 핵심 문제 요약 + 개선 우선순위 1~3위 + 대표 유저 발언 인용
- **대시보드 시각화** — 카테고리별 감성 도넛 차트 + 우선순위 카드
